## ✅ 필요 기능 목록 by @mcodnjs

1. 컴퓨터: 랜덤 숫자 생성
   - [ ] 1개의 랜덤 숫자 생성
   - [ ] 사이즈(3)만큼의 랜덤 숫자 생성
2. 플레이어: 숫자 입력
   - [ ] 플레이어 숫자 입력
   - [ ] 입력한 `String`을 `List` 타입으로 변경
3. 숫자 힌트 출력
   - [ ] ball
   - [ ] strike
   - [ ] nothing
4. 게임 재진행 혹은 종료
   - [ ] 플레이어 게임 재진행 여부 입력
   - [ ] 게임 재진행 혹은 종료
5. 예외 처리
   - [ ] 사용자가 잘못된 값을 입력할 경우
     - [ ] 숫자가 아닌 값을 입력할 경우
     - [ ] 숫자가 1~9로 구성되지 않은 경우
     - [ ] 입력한 숫자의 길이가 3이 아닌 경우
